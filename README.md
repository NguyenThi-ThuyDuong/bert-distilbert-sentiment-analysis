# bert-distilbert-sentiment-analysis

Deep learning project using BERT and DistilBERT for three-class sentiment analysis (positive, negative, neutral).





\# BERT vs DistilBERT Sentiment Analysis



This project implements sentiment classification using fine-tuned BERT and DistilBERT models.



\## Task

Three-class sentiment classification:

\- Positive

\- Negative

\- Neutral



\## Dataset

Balanced sentiment dataset in Excel format.



\## Models

Due to GitHub file size limitations, pretrained models are stored on Google Drive:

\- BERT model: https://drive.google.com/drive/folders/1RjpBYyTpgvQIkGVVxsatiogBNUDiq\_b8?usp=sharing

\- DistilBERT model: https://drive.google.com/drive/folders/11xm6D9J1OH6wj4NsQErbF00m8XRv4yV3?usp=sharing
